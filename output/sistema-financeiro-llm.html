Perfil de cada modelo
Econômico
Haiku 4.5

Latência: 1-3s. Bom para checagens, classificação, tarefas com instrução clara. 5x mais barato que Opus.

Equilibrado
Sonnet 4.6

Latência: 3-8s. Bom para conteúdo, código moderado, análise. 60% mais barato que Opus.

Premium
Opus 4.6

Latência: 5-15s. O melhor em raciocínio complexo, estratégia, copy avançada. O mais caro.

DONASYSTEM05
05 Regra de ouro

Quando usar cada modelo

Decisão rápida

Tarefa rotineira, checagem ou formatação → Haiku
Produção de conteúdo ou código moderado → Sonnet
Estratégia, decisão complexa ou copy de alto impacto → Opus
Não tem certeza → Sonnet (melhor custo-benefício)

TarefaModelo idealJustificativa
Heartbeat, health checkHAIKUInstrução simples, resposta binária
Backup, organização de memóriaHAIKUComandos de sistema, sem raciocínio
Watchdog, auditoriaHAIKUVerificação de lista, pouca criatividade
Posts, legendas, pesquisaSONNETConteúdo bom, não precisa de Opus
Código, scripts, automaçõesSONNETQualidade suficiente para a maioria
Geração de imagens (prompts)SONNETPrompts não exigem raciocínio profundo
