02 · Skills e Integrações

Ferramentas que o sistema usa

Cada skill é uma capacidade específica. O sistema escolhe automaticamente qual usar dependendo do que eu peço.

Canva Connect
Design · Exportação
Acessa a API do Canva. Lista designs, exporta em PNG/PDF, faz upload de assets e preenche templates automaticamente.
Whisper (OpenAI)
Transcrição de Áudio
Transcreve qualquer áudio em texto com precisão. Funciona com arquivos locais ou URLs públicas.
Geração de PDF
Documentos · Materiais
Cria PDFs profissionais a partir de HTML com o Design System ISIS v7. Ebooks, relatórios, apresentações.
Video Frames
Análise de Vídeo
Extrai frames e clipes de vídeos. Útil pra analisar reels, criar thumbnails e revisar conteúdo gravado.
Instagram Analytics
Dados · Performance
Puxa posts e reels de qualquer perfil. Analisa o que está performando, identifica padrões de conteúdo.
YouTube Analytics
Dados · Pesquisa
Acessa vídeos de canais. Útil pra pesquisa de tendências e análise de concorrência.
Remoção de Fundo
Imagem · Edição
Remove fundo de imagens automaticamente. Pra fotos de produto, materiais de marketing, posts.
Text to Speech
Áudio · Voz
Converte texto em áudio. Útil pra protótipos de conteúdo em áudio e acessibilidade.
Pesquisa Web
Dados · Tendências
Busca informações em tempo real na internet. Tendências, referências, dados de mercado, notícias.
1Password
Segurança · Credenciais
Gerencia todas as senhas e chaves de API com segurança. Nada fica exposto em texto claro.
