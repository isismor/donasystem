09 · Diretrizes

Uso consciente

Este sistema é a voz visual de Isis Moreira. Cada desvio precisa ser uma escolha consciente, não um acidente.

Ouro
✓ Faça
✓ Máximo 3 elementos com ouro por tela
✓ Use sempre via var(--ouro), nunca #b4a68c direto
✓ Reserve para o que realmente importa na hierarquia
✕ Evite
✕ Ouro em mais de 3 elementos simultâneos
✕ Usar o valor fixo #b4a68c; sempre use var(--ouro)
✕ Combinar texto ouro com fundo ouro no mesmo bloco
Tipografia
✓ Faça
✓ DM Serif italic para títulos emocionais e manifestos
✓ IBM Plex Mono exclusivamente para labels, código e meta
✓ Mono eyebrow + serif title: é a assinatura ISIS
✕ Evite
✕ Serif em corpo de texto longo; reservar para display
✕ Mono em mais de 3 linhas seguidas fora de código
✕ Letter-spacing positivo em texto body
Escrita & Tom
✓ Faça
✓ Ponto, vírgula, dois-pontos, ponto e vírgula
✓ ✦ como único ornamento visual da marca
✓ Frases diretas, sem rodeios, com profundidade
✕ Evite
✕ Travessões em qualquer contexto de UI
✕ Emojis em labels, badges, títulos ou UI
✕ Termos em inglês no conteúdo visível ao usuário
