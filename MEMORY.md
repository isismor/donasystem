## Ciclo de Memória

1. **Sessão relevante** → criar `memory/YYYY-MM-DD.md` com registro raw
2. **A cada poucos dias** → consolidar notas nos topic files
3. **Antes de compactação** → extrair lições e decisões primeiro
4. **Revisão mensal** → deletar lições táticas vencidas (>30 dias)
