## Ciclo de Memoria

1. **Sessao relevante** → criar `memory/YYYY-MM-DD.md` com registro raw
2. **A cada poucos dias** → consolidar notas nos topic files
3. **Antes de compactacao** → extrair licoes e decisoes primeiro
4. **Revisao mensal** → deletar licoes taticas vencidas (>30 dias)
